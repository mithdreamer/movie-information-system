# Geliştirme Planı

## Faz 1 - Excel + API

- Excel'den film adlarını oku.
- TMDb API ile film bilgilerini çek.
- OMDb API varsa IMDb puanını ekle.
- Sonuçları Excel dosyasına yaz.

## Faz 2 - Hata Yönetimi ve Cache

- Aynı film tekrar sorgulanmasın.
- Basit JSON cache dosyası eklensin.
- Hata logları tutulsun.

## Faz 3 - SQLite

- Film bilgileri SQLite veritabanına kaydedilsin.
- Daha önce çekilen film veritabanından getirilsin.

## Faz 4 - Web Arayüzü

- Flask veya FastAPI ile basit arayüz hazırlansın.
- Film adı input alanı olsun.
- Sonuçlar ekranda gösterilsin.
- Excel'e aktar butonu eklensin.

## Faz 5 - Kişisel Site Entegrasyonu

- Proje kişisel web sitesindeki öğrenme projeleri bölümüne eklensin.
- Proje açıklaması, ekran görüntüsü ve GitHub linki gösterilsin.
